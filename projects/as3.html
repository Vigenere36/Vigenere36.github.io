Assignment 3: Bezier Subdivision
Will timestamp when extra credit was done after due date
Note: Object files don't work perfectly, only look good with high step size.
The program converts the object file passed in to 'temp.bez' and reads that.
Note 2: Object files work on my machine but not my partner's for some reason.
If running it on an object file doesn't work, try running on the 'temp.bez' that it outputs.
Keys: arrows, shift+arrows, +, -, 'w', 's' as expected
'a' switches between uniform and adaptive subdivision
'r' resets the object's orientation to its inital orientation'
'h' enables hidden line mode
Simple wireframe with uniform subdivison
./as3 test.bez 0.05
Simple wireframe with hidden line mode enabled (press 'h')
./as3 test.bez 0.05
Simple wireframe with adaptive subdivision
./as3 test.bez 0.001 -a
Adaptive subdivision with shading
./as3 test.bez 0.001 -a
Uniform subdivision wireframe
./as3 teapot.bez 0.05

Adaptive subdivision with flat shading (intentional low poly)
./as3 teapot.bez 0.1 -a
Adaptive subdivision with smooth shading (intentional low poly)
./as3 teapot.bez 0.1 -a
Adaptive subdivision with smooth shading (high poly)
./as3 teapot.bez 0.001 -a
